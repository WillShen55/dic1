# Will's Personal Space Dashboard ⚡

Welcome to my personal cockpit and dashboard! This project is a premium, responsive single-page web app built with vanilla HTML, CSS, and JavaScript.

## ✨ Features

- 🌌 **Glassmorphic Design**: Clean and modern backdrop-blur aesthetics with glowing gradient borders and glass panels.
- 🕒 **Time-Adaptive Theme**: The layout changes its dynamic ambient color scheme and welcome greeting based on the hour of the day (Morning, Afternoon, Evening, and Night).
- 🧭 **Day Tracker Telemetry**: Visualizes what percentage of the current day has elapsed using a custom circular SVG progress ring and calculates remaining hours in real-time.
- 🎯 **Interactive Cursor Spotlight**: A custom glowing light tracks the mouse cursor movements in the background for a smooth micro-interaction.
- 🛠️ **Tech & Skills Dock**: Elegant, glowing skill badges highlighting development languages and tools.
- 📱 **Fully Responsive**: Optimized to provide a perfect viewing experience on mobile devices, tablets, and desktops.

## 📂 File Structure

```text
├── index.html   # Main structure & layout
├── style.css    # Typography, glassmorphism UI, & background animations
├── script.js    # Clock engine, telemetry calculations, & mouse-tracking script
├── avatar.png   # Futuristic AI-generated profile avatar
└── README.md    # Project documentation
```

## 🚀 How to Run Locally

Since this app is built purely on frontend technologies, there is no setup required:
1. Clone or download this repository.
2. Open `index.html` directly in any modern web browser.
3. Enjoy the live dashboard!
